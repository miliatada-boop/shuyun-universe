## 一句话气质

Shuyun Universe 是一种安静、有深度、自由流动的精神生活文明。它不是热闹的疗愈感，也不是商业化的灵性包装，而是：清醒、温柔、留白、国际化、带一点东方漆器和旧金属的克制质感。

关键词：

- Flow / 流
- Gathering / 聚
- Atmosphere / 氛围
- Gentle Power / 柔力
- Borderless Living / 无界
- Spiritual Ecology / 精神生态

## 色彩系统

### 核心底色

| 名称 | 色值 | 用法 |
| --- | --- | --- |
| Chalk | `#F5F2EE` | 主背景、留白、图文底色 |
| Sand Veil | `#E8E2D8` | 次背景、分区、柔和渐变 |
| Warm Drift | `#C4B49A` | 纸张、器物、温暖中性色 |
| Mist | `#9EA8B2` | 小字、注释、弱信息 |
| Sage Tide | `#7B8E82` | 温柔自然感、生活方式内容 |
| Deep Current | `#4A5C6B` | 正文、次标题、沉静蓝灰 |
| Abyssal | `#1C2B35` | 主标题、深色底、品牌压舱石 |

### 暖色点缀

暖色只能像印章、金线、漆器边缘一样使用，不要大面积铺满。

| 名称 | 色值 | 用法 |
| --- | --- | --- |
| Lacquer | `#6B2737` | 重要副标题、细线、强调词 |
| Cinnabar | `#9B3A40` | 少量暖红点缀 |
| Antique Gold | `#B8955A` | 细分割线、印章、仪式感细节 |
| Pale Gold | `#D4BA7E` | 深色底上的温和金色文字 |

### 推荐组合

- 高级浅底：`Chalk + Sand Veil + Abyssal + Lacquer + Antique Gold`
- 深海仪式感：`Abyssal + Deep Current + Chalk + Pale Gold`
- 温柔生活方式：`Chalk + Sage Tide + Mist + Deep Current`
- 活动物料：`Chalk + Warm Drift + Antique Gold + Lacquer`

## 字体系统

### 英文字体

- Serif：`Cormorant Garamond`，本机可用 `Didot` 或 `Baskerville` 代替。
- Sans：`DM Sans`，本机可用 `Avenir Next` 代替。

### 中文字体

- 海报主标题：使用干净、厚一点的黑体或现代宋黑感字体，确保小红书缩略图可读。
- 正文/小字：细一点的无衬线，保持安静和国际化。
- 不要使用可爱、圆润、手写、综艺感字体。

### 层级

- 主标题：最大、最清楚，一到两行。
- 副标题：只解释核心情绪，不超过一行。
- 来源信息：很小，作为背景信息存在。
- 英文标签：可以拉开字距，但只做气质，不抢主题。

## 版式原则

- 留白要多，至少让画面有 30% 的安静空间。
- 一屏只突出一个观点。
- 金色和漆红只出现一两次。
- 避免复杂卡片、密集按钮、过多边框。
- 分割线使用 1px 细线，偏金或沙色。
- 渐变要像自然光，不要像色块横切。
- 图片与文字不能无意重叠；如果叠加，必须有明确的雾化或暗化处理。

## 海报 / 小红书封面参考

### 默认封面模板

尺寸：`1080x1440`

结构：

1. 顶部少量品牌信息：`SHUYUN UNIVERSE`、人物名或主题名。
2. 中上部放主体图，优先选择清晰人物/空间/物件。
3. 图片底部做自然雾化，让它融入背景。
4. 主标题放在下半部，最大，一到两行。
5. 底部只保留很小的来源信息或英文标题。

推荐配色：

- 背景：顶部 `Sand Veil`，下方 `Chalk`，中间长渐变。
- 主标题：`Abyssal`
- 副标题：`Lacquer`
- 细线：`Antique Gold`
- 小字：`Deep Current` 或 `Mist`

文案规则：

- 主标题最多两行。
- 不要在封面上放完整摘要。
- “谁的视频 / 主题 / 来源”可以有，但必须小。
- 左上角文字不要压到图片。

当前验证过的封面方向：

```text
真正的平静
不推开，也不抓取
```

## 视频视觉参考

### 视频封面

视频封面可以沿用小红书海报模板，但要更考虑移动端缩略图：

- 主标题更大，最多两行。
- 人物或主体更明显。
- 来源信息更小。
- 不要用太多英文小字。
- 如果是精神成长/冥想内容，优先使用浅底或深海底，不要使用高饱和色。

### 字幕

默认字幕视觉：

- 中文在上，英文在下。
- 中文用品牌暖色系：优先 `Lacquer #6B2737`、柔和玫瑰粉、或用户指定色。
- 英文用白色。
- 两者都加黑色外边框，保证可读。
- 字体要干净，避免花哨。
- 只加字幕，不额外加贴纸、边框、标题条，除非用户要求。

### 视频内容包装

- 开头不强行加大 logo。
- 不做综艺感转场。
- 可用极轻的片头/片尾：深海底、细金线、serif 标题。
- 视频整体感觉：安静、克制、可收藏。

## 网页参考

网页展示版来自 `brand-identity.html`。

网页方向：

- Hero 可用深海底 `Abyssal`，文字用 `Chalk`。
- 背景可以有非常柔和的径向光感，但不能有明显炫彩渐变。
- 导航最多 4 项。
- 一页一个主行动。
- 页面像杂志，不像 SaaS 营销页。
- 不要弹窗、倒计时、强促销 banner。
- 动画只用淡入、轻微位移，不用弹跳和夸张滑动。

网页组件：

- Label：小写或大写英文，字距拉开，颜色用 `Mist`。
- 标题：serif，轻、慢、呼吸感。
- 正文：sans，行距大，颜色用 `Deep Current`。
- 按钮：深色底或细边框，少量使用。
- 卡片：尽量少，用于内容块而不是装饰。

## 图片 / 摄影方向

关键词：

- Unbleached linen
- Handmade paper
- Weathered stone
- Still water
- Frosted glass
- Ceramic
- Bone
- Driftwood
- Heavyweight cotton
- Cold metal

光线：

- 阴天北欧光
- 黎明前蓝调时刻
- 柔和午后光
- 冬日低角度阳光

构图：

- 至少 40% 留白。
- 物体少，但有质感。
- 不要网红滤镜、过曝粉调、强烈 bokeh。
- 不要 motivational quote 直接压风景图。

## 文案声音

是：

- Precise / 精准
- Quiet / 安静
- Warm / 温暖
- Philosophical / 有思想
- Grounded / 落地
- Curious / 好奇
- Unhurried / 不急
- Specific / 具体
- Poetic / 有诗意

不是：

- Urgent / 急促
- Loud / 吵
- Coach-y / 教练腔
- Promotional / 强促销
- Vague / 空泛
- Trendy / 追热点
- Performative / 表演感

中文文案要像一个清醒的人在轻声说话，不要像课程广告，也不要像鸡汤。

## 使用前检查

交付任何视觉前，快速检查：

- 是否一眼能看出主标题？
- 是否只突出一个主题？
- 暖色是否只作为点缀？
- 渐变是否自然？
- 图片和文字是否没有意外重叠？
- 有没有过多小字？
- 是否有足够留白？
- 是否像 Shuyun Universe，而不是普通小红书模板？
