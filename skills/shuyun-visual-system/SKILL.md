Use this skill whenever an output should look like Shuyun Universe.

## Workflow

1. Identify the output type: poster, video cover, subtitles, webpage, document, deck, social post, or event material.
2. Read the relevant section in the central VI file.
3. Apply the palette, typography, spacing, image treatment, and copy tone.
4. Keep the design quiet, warm, premium, and spacious.
5. Check for visual mistakes: crowded text, harsh gradients, accidental overlap, overuse of red/gold, or generic template styling.

## Default Direction

- Use Chalk/Sand as calm bases.
- Use Abyssal/Deep Current for strong text.
- Use Lacquer/Antique Gold only as small meaningful accents.
- Make one idea dominant.
- Keep source details small.
- Prefer editorial stillness over marketing loudness.

## Relationship To Other Skills

When another skill creates visual assets, such as `$youtube-xhs-localization`, combine that workflow with this visual system. This skill supplies the look; the other skill supplies the production steps.
