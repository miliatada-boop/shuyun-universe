# 抒云内容素材库

> Shuyun Content Library — 所有内容产出的原料池与归档系统

## 结构说明

```
content/
├── 01-原料池/     → 你扔进来的任何素材，按类型分类
├── 02-加工区/     → 正在写的内容草稿
├── 03-待发库/     → 写好了等排期发布的内容
└── 04-已发归档/   → 已发布内容存档
```

## 原料池分类

| 文件 | 收什么 |
|------|--------|
| 灵感碎片 | 突然的想法、金句、概念 |
| 观察记录 | 城市、空间、品牌现场感受 |
| 人物印记 | 见了谁、聊了什么、触动 |
| 事件复盘 | 经历或组织的事件 |
| 对标研究 | 值得参考的内容/品牌 |
| 内在感悟-冥想 | 内在洞察、精神状态 |
| 客户洞察 | 工作中的判断（脱敏） |
| 案例素材 | 项目片段（脱敏） |

## 使用方式

1. 有素材 → 扔进 `01-原料池` 对应文件
2. 要写内容 → 打开 Claude Project「策略师」提炼选题
3. 确认选题 → 打开「公域写手」或「私域运营」生产内容
4. 写好的稿 → 放进 `03-待发库`
5. 发出去后 → 移入 `04-已发归档`
